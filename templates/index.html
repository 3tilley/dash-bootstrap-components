Dash Bootstrap Components

Easy to use layout and components for Plotly Dash

GitHub Components

Installation

Install dash-bootstrap-components via pip: 

pip install dash-bootstrap-components

Getting started

dash-bootstrap-components relies on Twitter Bootstrap. To use this package, inject the Bootstrap stylesheet into your application: 

external_stylesheets = ["https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css"]

app = dash.Dash(__name__, external_stylesheets=external_stylesheets)
